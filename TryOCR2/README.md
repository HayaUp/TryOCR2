# C#, WPFで画像から文字認識を試す

## 開発環境など

- Microsoft Visual Studio Community 2019
  - Version 16.11.26
- C#
- WPF
- .NET5

## 大まかな処理

画像内の文字を認識したい

- 画像を選ぶ
- 選んだ画像を表示する
- 選んだ画像から文字を認識する
- 認識した文字を表示する
- 認識した文字をクリップボードにコピーする
